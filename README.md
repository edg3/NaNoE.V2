# NaNoE.V2

This is the new version of NaNoE, more details available on: https://edg3.co.za/ [Updates coming soon there]

The main changes:
- Moved to C# WPF
- Added 'Bookmarks'
- Added 'Notes'
- Adjusted 'Helpers'
- Changed the editing entirely for ease of access
- Allowed flagging paragraphs for edit 'ignore'
- Allowed 'reset flags'
